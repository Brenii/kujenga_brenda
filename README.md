# kujenga_brenda
Submission for my final project of the Kujenga project by coRE AI 2026. Includes Jupiter notebook, README, and datasets used.

To run through the the code, ensure you have a python environment set up on your computer and adjust the code to reflect the appropriate file path for reading the datasets. Alternatively upload the jupiter notebook to your google drive and use google Colab. Make sure to upload your datasets as well.

I use two separate datasets from worldbank for the World Development Indicators. My project is centred around investigating why Kenya continues to experience high levels of poverty(~39%) despite a growth in the GDP. My argument is that GDP growth is relevant when looking at the high level macro-economy but other factors must be accounted for at the micro-economic level to justify the economic challenges that ordinary Kenyans face like high taxes, high cost of food, and poor public services like education and healthcare. See the random forest regression for details on the rank of indicators that are likely to predict povert levels based on the Kenyan data provide. 

# Challenges
My datasets were small and had significant missing values and therefore this analysis can only be used for exploratory purposes. It does not reflect conclusions that can be used for decision-making.
